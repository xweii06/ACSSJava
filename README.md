# APU Car Sales System (ACSS) 

Java-based desktop application developed using basic **Java Swing (GUI)**. 

The system is designed to manage and streamline car sales operations by providing structured modules 
for managing staff, salesmen, customers, cars, payments, feedback, and reporting.
The system supports role-based access control and is built to improve efficiency, data organization, and transparency in car sales management processes.

## Key Features

- Role-based access control (Admin, Salesman, Customer)
- Full CRUD operations for staff, cars, and users
- Secure login with lockout mechanism
- File handling for persistent data storage
- Image upload and display for car listings
- Feedback and rating system
- Reporting and analytics dashboard
